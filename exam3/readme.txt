Running steps:
======================================================================
For running the algorithm with one dimensional objective function
1. Remove/add the "%" symbol to enable/disable the function preferred
   in the ObjectiveFunction1_Sasarak_Sun.m
2. All parameters can be changed at main1_Sasarak_Sun.m
3. Run main1_Sasarak_Sun

======================================================================
For running the algorithm with two dimensional objective function
1. Remove/add the "%" symbol to enable/disable the function preferred
   in the ObjectiveFunction2_Sasarak_Sun.m
2. All parameters can be changed at main2_Sasarak_Sun.m
3. Run main2_Sasarak_Sun
